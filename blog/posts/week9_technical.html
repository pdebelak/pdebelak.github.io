Recursion

Recursion is a process where one of the steps of the process involves performing the process itself. The most common one we come across is "lather, rinse, repeat." Obviously, that would mean washing your hair for all eternity.

How do we use it?

In programming, recursion is quite useful. Often a method will include a call to itself in its definition. When a programmer says "recursion", that is what she likely means.

Example

Let's use an example from a challenge I recently worked on. The goal was to write a method that took an integer as input and outputed that integer as English words. So, if the input was 199 the output should be "one hundred ninety nine". My basic solution was this:

  def in_words(num)
    ones_teens = ["one", "two", "three", "four", "five", "six", "seven", "eight", "nine", "ten", "eleven", "twelve", "thirteen", "fourteen", "fifteen", "sixteen", "seventeen", "eighteen", "nineteen"]
    tens = ["twenty", "thirty", "forty", "fifty", "sixty", "seventy", "eighty", "ninety"]
    if num < 20
      ones_teens[num - 1]
    else
      words = []
      if num / 100 > 0 && num != 0 then words << "#{ones_teens[(num / 100) - 1]} hundred" end
      num -= (num / 100 * 100)
      if num / 10 > 1 && num != 0 then words << tens[(num / 10) - 2] end
      num -= (num / 10 * 10)
      if num != 0 then words << ones_teens[num - 1] end
      words.join(" ")
    end
  end

This method works for numbers from 1 to 999. If I want to extend past 1,000, though, I can use recursion to easily add a bunch of functionality. Using recursion, a few more lines of code allowed me to quickly extend it to work for numbers from 1 to 999,999,999,999,999:

  def in_words(num)
    ones_teens = ["one", "two", "three", "four", "five", "six", "seven", "eight", "nine", "ten", "eleven", "twelve", "thirteen", "fourteen", "fifteen", "sixteen", "seventeen", "eighteen", "nineteen"]
    tens = ["twenty", "thirty", "forty", "fifty", "sixty", "seventy", "eighty", "ninety"]
    if num < 20
      ones_teens[num - 1]
    else
      words = []
      if num / 1000000000000 > 0 && num != 0 then words << "#{in_words(num/1000000000000)} trillion" end
      num -= (num / 1000000000000 * 1000000000000)
      if num / 1000000000 > 0 && num != 0 then words << "#{in_words(num/1000000000)} billion" end
      num -= (num / 1000000000 * 1000000000)
      if num / 1000000 > 0 && num != 0 then words << "#{in_words(num/1000000)} million" end
      num -= (num / 1000000 * 1000000)
      if num / 1000 > 0 && num != 0 then words << "#{in_words(num/1000)} thousand" end
      num -= (num / 1000 * 1000)
      if num / 100 > 0 && num != 0 then words << "#{ones_teens[(num / 100) - 1]} hundred" end
      num -= (num / 100 * 100)
      if num / 10 > 1 && num != 0 then words << tens[(num / 10) - 2] end
      num -= (num / 10 * 10)
      if num != 0 then words << ones_teens[num - 1] end
      words.join(" ")
    end
  end

That is the power of recursion. Pretty cool, right?
