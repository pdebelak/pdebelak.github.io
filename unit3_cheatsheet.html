Ruby Array Cheatsheet

Stop being a cheater and click here for the real deal.

Create an array

Literal constructor:

array = ["one", 2, [3, "four], {"five" => 6}, 7]

Call #new:

array = Array.new

With arguments:

Array.new(3) == [nil, nil, nil]

That's how you make an array!

Access array elements

Let's take the array array = [1, 2, 3, 4, 5] and access it!

By index:

array[2] == 3
array[-2] == 4

Other tricks

array.first == 1
array.last == 5

That's how you access what is in the array!

Add to the array

Add at the end:

array.push(6)
array == [1, 2, 3, 4, 5, 6]

Also, array.push(6) is the same as array << 6.

Add to the beginning:

array.unshift(0)
array == [0, 1, 2, 3, 4, 5, 6]

That's how you add to an array!

Remove from the array

array = [1, 2, 2, 3, 4, 5]

Remove and return the last item:

array.pop #returns 5
array == [1, 2, 2, 3, 4]

Remove and return the first itme:

array.shift #returns 1
array == [2, 2, 3, 4]

Delete by index:

array.delete_at(3)
array == [2, 2, 3]

Delete by contents:

array.delete(2)
array == [3]

That's how you delete items from an array!
